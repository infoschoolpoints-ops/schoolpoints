## אייקונים נדרשים

### 1. עמדת ניהול (admin.ico) 👨‍💼
**קונצפט:** מנהל/מורה עם כוכב או לוח ניקוד

**אלמנטים מוצעים:**
- כוכב זהב מרכזי
- לוח/טבלה ברקע
- דמות מנהל/מורה (אופציונלי)
- צבעים: כחול + זהב

**סגנון:** מקצועי, רשמי, מייצג סמכות

**קובץ:** `icons/admin.ico`

---

### 2. עמדה ציבורית (public.ico) 👨‍🎓
**קונצפט:** תלמיד/משתמש עם כרטיס או נקודות

**אלמנטים מוצעים:**
- דמות תלמיד פשוטה
- כרטיס/תג זהות
- כוכבים קטנים (נקודות)
- צבעים: כחול + ירוק

**סגנון:** ידידותי, נגיש, צעיר

**קובץ:** `icons/public.ico`

---

### 3. עמדת קופה (cashier.ico) 🛒
**קונצפט:** עגלת קניות או מטבעות עם כוכבים

**אלמנטים מוצעים:**
- עגלת קניות
- מטבעות/נקודות
- סימן שקל (₪) או כוכב
- צבעים: כתום + זהב

**סגנון:** מסחרי, ברור, מזמין

**קובץ:** `icons/cashier.ico`

---

### 4. מתקין (installer.ico) 📦
**קונצפט:** לוגו ראשי של המערכת

**אלמנטים מוצעים:**
- כוכב גדול במרכז
- טקסט "SP" או "נקודות"
- מסגרת/מעטפת
- צבעים: כחול + זהב + ירוק

**סגנון:** מייצג, מקצועי, זיהוי המותג

**קובץ:** `icons/installer.ico`

## כלים מומלצים ליצירת אייקונים

### תוכנות חינמיות:
1. **GIMP** - עורך תמונות חינמי
   - תומך ביצירת קבצי ICO
   - Plugin: GIMP ICO Plugin

2. **Inkscape** - עורך וקטורי חינמי
   - מצוין לעיצוב אייקונים
   - ייצוא ל-PNG ואז המרה ל-ICO

3. **IcoFX** (גרסה חינמית) - עורך אייקונים ייעודי
   - תומך בכל הגדלים
   - ממשק פשוט

### כלים אונליין:
- **Favicon.io** - יצירת אייקונים מטקסט/אימוג'י
- **RealFaviconGenerator** - המרה ויצירת אייקונים
- **ConvertICO** - המרת PNG/JPG ל-ICO

## תהליך יצירה מומלץ

### שלב 1: עיצוב ב-256x256
1. צור תמונה בגודל 256x256 פיקסלים
2. השתמש ברקע שקוף (PNG)
3. שמור כ-PNG באיכות גבוהה

### שלב 2: יצירת גדלים נוספים
1. שכפל את התמונה
2. שנה גודל ל-128, 64, 48, 32, 16
3. התאם פרטים קטנים בגדלים הקטנים (פשט אם צריך)

### שלב 3: המרה ל-ICO
1. השתמש בכלי המרה (IcoFX, GIMP, או אונליין)
2. שלב את כל הגדלים לקובץ ICO אחד
3. שמור בתיקיית `icons/`

## אינטגרציה במערכת

לאחר יצירת האייקונים, יש לעדכן:

### קבצי .spec (PyInstaller)
```python
# SchoolPoints_Admin.spec
exe = EXE(
    ...
    icon='icons/admin.ico',
    ...
)

# SchoolPoints_Public.spec
exe = EXE(
    ...
    icon='icons/public.ico',
    ...
)

# SchoolPoints_Cashier.spec
exe = EXE(
    ...
    icon='icons/cashier.ico',
    ...
)
```

### קובץ Installer (SchoolPoints_Installer.iss)
```ini
[Setup]
SetupIconFile=icons\installer.ico
UninstallDisplayIcon={app}\Admin\SchoolPoints_Admin.exe

[Icons]
Name: "{group}\עמדת ניהול"; Filename: "{app}\Admin\SchoolPoints_Admin.exe"; IconFilename: "{app}\icons\admin.ico"
Name: "{group}\עמדה ציבורית"; Filename: "{app}\Public\SchoolPoints_Public.exe"; IconFilename: "{app}\icons\public.ico"
Name: "{group}\עמדת קופה"; Filename: "{app}\Cashier\SchoolPoints_Cashier.exe"; IconFilename: "{app}\icons\cashier.ico"
```

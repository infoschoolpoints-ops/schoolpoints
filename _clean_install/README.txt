========================================
תיקיית קבצים נקיים להתקנה
========================================

תיקייה זו מכילה קבצים נקיים ללא נתונים אישיים, שישמשו לבניית חבילת ההתקנה.

קבצים בתיקייה:
-----------------
✓ school_points.db          - מסד נתונים ריק (ללא תלמידים/מורים)
✓ config.json               - הגדרות בסיסיות (ללא נתיבים אישיים)
✓ bonus_settings.json       - הגדרות בונוס בסיסיות
✓ color_settings.json       - טווחי צבעים (ברירת מחדל)
✓ db_path.txt              - קובץ הגדרות נתיב DB (רק הסברים)
✓ excel_path.txt           - קובץ הגדרות נתיב Excel (רק הסברים)
✓ master_card.txt          - כרטיס מאסטר (ריק)

שימוש:
-------
קבצים אלו משמשים את PyInstaller דרך קבצי ה-.spec:
- SchoolPoints_Admin.spec
- SchoolPoints_Public.spec

כאשר בונים את ה-EXE, הקבצים הנקיים האלו מוכללים במקום הקבצים 
המכילים נתונים אישיים שנמצאים בתיקייה הראשית.

⚠️ אל תמחק תיקייה זו - היא נדרשת לבניית התוכנה!

יש לך 2 אפשרויות להפעלה (בחר את המועדף עליך):

אפשרות 1️⃣: קבצי .pyw (הכי פשוט!)
─────────────────────────────────────────
✅ admin_station.pyw     - עמדת ניהול
✅ public_station.pyw    - עמדה ציבורית

פשוט לחיצה כפולה על הקובץ והתוכנה נפתחת!
✅ אין חלון CMD בכלל
✅ מסך טעינה יפה עם "אנא המתן..."
✅ אייקון 🎓 חמוד


אפשרות 2️⃣: קבצי .bat (כמו קודם, אבל ללא CMD)
─────────────────────────────────────────
✅ run_admin.bat         - עמדת ניהול
✅ run_public.bat        - עמדה ציבורית

פשוט לחיצה כפולה על הקובץ והתוכנה נפתחת!
✅ אין חלון CMD (הוא נסגר מיד)
✅ מסך טעינה יפה


המלצה: 👉 השתמש בקבצי .pyw - הם הכי נקיים!

🎬 מה קורה כשפותחים את התוכנה?
═══════════════════════════════════════════════════════════════════

1. לחיצה כפולה על הקובץ (.pyw או .bat)

2. מופיע מסך טעינה יפה:
   ┌─────────────────────────────────────┐
   │                                     │
   │             🎓                      │
   │                                     │
   │      מערכת ניהול נקודות            │
   │                                     │
   │   אנא המתן, התוכנה עולה...         │
   │                                     │
   │   ▓▓▓▓▓▓▓▓▓░░░░░░░░░               │
   │                                     │
   │      מאתחל מערכת...                │
   └─────────────────────────────────────┘

3. אחרי שניה-שתיים, התוכנה נפתחת!

4. אין חלון שחור (CMD) בכלל! 🎉

═══════════════════════════════════════════════════════════════════
🖥️ יצירת קיצורי דרך לשולחן העבודה
═══════════════════════════════════════════════════════════════════

כדי ליצור אייקון יפה בשולחן העבודה:

1. לחץ לחיצה ימנית על admin_station.pyw
2. בחר "שלח אל" → "שולחן העבודה (קיצור דרך)"
3. שנה את שם הקיצור ל: "⭐ עמדת ניהול"

4. לחץ לחיצה ימנית על public_station.pyw
5. בחר "שלח אל" → "שולחן העבודה (קיצור דרך)"
6. שנה את שם הקיצור ל: "🎓 עמדת תלמידים"

עכשיו יש לך 2 אייקונים יפים בשולחן העבודה!
